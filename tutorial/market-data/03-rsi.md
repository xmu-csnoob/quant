# RSI (Relative Strength Index)

> 相对强弱指标，判断超买超卖的振荡指标，由 J. Welles Wilder Jr. 发明

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **超买超卖判断**：价格上涨过多会回调，下跌过多会反弹
- **多空力量对比**：RSI > 50 = 多方强，RSI < 50 = 空方强
- **提前预警**：RSI背离常领先于价格反转

### 解决什么问题
- 不知道何时价格"涨太多"或"跌太多"
- 需要判断当前是多头还是空头占优
- 趋势指标（MA/MACD）在震荡市失效，需要补充

### 适用场景
- 震荡市：高抛低吸的主要指标
- 趋势市：判断回调/反弹的结束位置
- 识别极端行情：超买区提示风险，超卖区提示机会

### 指标形态

```
RSI 值域：0 - 100

    70 ───────── 超买区（可能回调）
    |
    |    正常波动区
  50 ───────── 多空分界线
    |
    |
    30 ───────── 超卖区（可能反弹）
    |
```

## 二、数学公式（如何计算？）

### 计算步骤

**1. 计算涨跌幅**

对于每一天，计算收盘价变化：
```
涨跌 = 今日收盘价 - 昨日收盘价
```

**2. 分离涨跌**

```
上涨幅度 = max(涨跌, 0)      # 上涨时为正值，下跌时为0
下跌幅度 = max(-涨跌, 0)     # 下跌时为正值，上涨时为0
```

**3. 计算平均涨跌幅（使用EMA平滑）**

```
平均涨幅 = EMA(上涨幅度, 周期)
平均跌幅 = EMA(下跌幅度, 周期)
```

**4. 计算相对强度（RS）**

```
RS = 平均涨幅 / 平均跌幅
```

**5. 计算RSI**

```
RSI = 100 - (100 / (1 + RS))

或者等价公式：

        平均涨幅
RSI = ─────────────── × 100
      平均涨幅 + 平均跌幅
```

### 参数说明

| 参数 | 标准值 | 可调范围 | 说明 |
|------|--------|----------|------|
| 周期 | 14 | 9-25 | 14是Wilder推荐值 |
| 超买线 | 70 | 70-80 | 越高信号越少但越准确 |
| 超卖线 | 30 | 20-30 | 越低信号越少但越准确 |

### 计算示例

假设14天数据（简化版）：

```
天数 | 收盘价 | 涨跌 | 上涨幅度 | 下跌幅度
-----|--------|------|----------|----------
 1   | 10.00  | -    | 0        | 0
 2   | 10.50  | +0.5| 0.50     | 0
 3   | 11.00  | +0.5| 0.50     | 0
 4   | 10.80  | -0.2| 0        | 0.20
 5   | 10.60  | -0.2| 0        | 0.20
...  | ...    | ... | ...      | ...
14   | 11.50  | +0.3| 0.30     | 0

假设：
平均涨幅 = 0.40
平均跌幅 = 0.15

RS = 0.40 / 0.15 = 2.67

RSI = 100 - (100 / (1 + 2.67))
    = 100 - (100 / 3.67)
    = 100 - 27.2
    = 72.8

→ RSI = 72.8，进入超买区
```

## 三、应用（如何使用？）

### 1. 基础信号

| RSI值 | 状态 | 操作思路 |
|-------|------|----------|
| > 70 | 超买 | 考虑卖出/做空 |
| 50-70 | 强势 | 持股/逢低买入 |
| 30-50 | 弱势 | 持币/逢高卖出 |
| < 30 | 超卖 | 考虑买入/做多 |

### 2. 超买超卖信号

**超买区信号（卖出）**：
```
RSI进入70以上 → 警惕
RSI从70以上跌破70 → 卖出信号

示例：
RSI: 75 → 78 → 72 ↘ 68
         ↑      ↑
      警惕   卖出！
```

**超卖区信号（买入）**：
```
RSI进入30以下 → 关注
RSI从30以下突破30 → 买入信号

示例：
RSI: 25 → 22 → 28 ↗ 32
         ↑      ↑
      关注   买入！
```

### 3. 背离信号（重要！）

**顶背离（看跌）**：
```
价格：           ↑ 创新高
RSI：            ↗ ↘  未创新高
                  |
                 背离！可能下跌
```

**底背离（看涨）**：
```
价格：           ↘ 创新低
RSI：            ↘ ↗  未创新低
                  |
                 背离！可能上涨
```

### 4. 中轴策略（RSI=50）

```
RSI突破50向上 → 多头占优，买入信号
RSI跌破50向下 → 空头占优，卖出信号

应用：
- 趋势确认工具
- 判断当前市场强弱
```

### 5. 形态分析

RSI也可以形成K线类似的形态：

| 形态 | 含义 | 信号 |
|------|------|------|
| 双底（W底） | RSI在30附近两次探底回升 | 买入 |
| 双顶（M头） | RSI在70附近两次冲顶回落 | 卖出 |
| 上升三角形 | RSI高位震荡向上突破 | 继续持有 |
| 下降三角形 | RSI低位震荡向下突破 | 观望 |

### 6. 多周期共振

最佳信号：
```
日线RSI < 30
周线RSI < 30
月线RSI 在40附近（不是极端）

→ 日线、周线同时超卖，买入机会！
```

### A股实战注意事项

1. **T+1影响**：
   - 当天RSI超买不能卖出
   - 需要提前判断：RSI接近70时考虑卖出

2. **涨跌停限制**：
   - 涨停股票RSI会持续在超买区钝化
   - 不要因RSI高位而错过强势股

3. **A股特性**：
   - 龙头股可以RSI长期>80
   - ST股可能RSI长期<20
   - 政策利好会瞬间改变RSI形态

4. **参数调整建议**：
   - 大盘股：RSI(14)标准参数
   - 小盘股：RSI(9)更敏感
   - 波动大的股票：RSI(21)更稳定

### 常见陷阱

| 陷阱 | 说明 | 应对 |
|------|------|------|
| 强势钝化 | 牛股RSI长期>70，一直等不到卖出 | 结合K线和成交量，不要单看RSI |
| 弱势钝化 | 熊股RSI长期<30，反弹就卖出 | 趋势明确时不用等RSI回升 |
| 震荡市假信号 | RSI在30-70之间频繁波动 | 震荡市结合其他指标 |
| 单一指标依赖 | 只看RSI容易买入"下跌中的飞刀" | 必须结合趋势指标（MA） |
