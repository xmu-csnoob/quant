# KDJ (随机指标 / Stochastic Oscillator)

> 由 George Lane 发明，A股最常用的短期超买超卖指标

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **相对位置**：价格在近期区间的相对位置
- **超买超卖**：价格接近区间高点=超买，接近低点=超卖
- **短期敏感**：比RSI更敏感，适合短线交易

### 解决什么问题
- RSI反应太慢，错过短线机会
- 需要更灵敏的超买超卖信号
- 需要捕捉短期高低点

### 适用场景
- 短线波段操作
- 震荡市高抛低吸
- 判断短期超买超卖
- 捕捉短线反转点

### 指标构成

```
KDJ由三条线组成：

K线 = 快速随机值
D线 = K线的平滑线
J线 = K线的偏离值

图形：
100 ─────────────────
  |     ↗ K线（最快）
  |    ↗
  |   ↗  D线（中速）
70 |  ↗
  | ──── J线（最快，超范围）
  |
50 ─────────────────
  |
30 |
  |
  |
 0 ──────────────────
```

## 二、数学公式（如何计算？）

### 计算步骤

**1. 计算RSV（未成熟随机值）**

```
RSV = (今日收盘价 - N日内最低价) / (N日内最高价 - N日内最低价) × 100%

标准参数：N = 9
```

**2. 计算K值**

```
K值 = 2/3 × 昨日K值 + 1/3 × 今日RSV

初始K值 = 50
```

**3. 计算D值**

```
D值 = 2/3 × 昨日D值 + 1/3 × 今日K值

初始D值 = 50
```

**4. 计算J值**

```
J值 = 3 × 今日K值 - 2 × 今日D值

或者：
J值 = 3 × K - 2 × D
```

### 参数说明

| 参数 | 标准值 | 可调范围 | 说明 |
|------|--------|----------|------|
| RSV周期 | 9 | 5-20 | 越小越敏感 |
| K平滑系数 | 3 | 固定 | 通常不改 |
| D平滑系数 | 3 | 固定 | 通常不改 |

### 计算示例

```
假设9天内：
最高价 = 15.00
最低价 = 10.00
今日收盘价 = 13.00

步骤1：计算RSV
RSV = (13.00 - 10.00) / (15.00 - 10.00) × 100%
    = 3.00 / 5.00 × 100%
    = 60%

步骤2：计算K值（假设昨日K=55）
K = 2/3 × 55 + 1/3 × 60
  = 36.67 + 20
  = 56.67

步骤3：计算D值（假设昨日D=53）
D = 2/3 × 53 + 1/3 × 56.67
  = 35.33 + 18.89
  = 54.22

步骤4：计算J值
J = 3 × 56.67 - 2 × 54.22
  = 170.01 - 108.44
  = 61.57

结果：K=56.67, D=54.22, J=61.57
```

## 三、应用（如何使用？）

### 1. 基础信号

| KDJ状态 | 市场含义 | 操作建议 |
|---------|----------|----------|
| K > 80 | 超买 | 考虑卖出 |
| K < 20 | 超卖 | 考虑买入 |
| D > 80 | 严重超买 | 减仓 |
| D < 20 | 严重超卖 | 建仓 |
| J > 100 | 极度超买 | 强烈卖出信号 |
| J < 0 | 极度超卖 | 强烈买入信号 |

### 2. 金叉与死叉

**金叉（买入信号）**：
```
条件：
1. K线上穿D线
2. 在20以下低位金叉更强
3. J线同时向上

图形：
      ↗ K
    ↗
  ── D
  20

信号强度：
低位金叉（K<20） > 中位金叉 > 高位金叉（K>80，可能骗线）
```

**死叉（卖出信号）**：
```
条件：
1. K线下穿D线
2. 在80以上高位死叉更强
3. J线同时向下

图形：
  ── D
    ↘
      ↘ K
  80

信号强度：
高位死叉（K>80） > 中位死叉 > 低位死叉（K<20，可能骗线）
```

### 3. 超买超卖区间

```
超买区（K > 80）：
- 可能回调
- 但强势股可以持续超买
- 等死叉确认再卖出

超卖区（K < 20）：
- 可能反弹
- 但弱势股可以持续超卖
- 等金叉确认再买入

正常区间（20 < K < 80）：
- 观望
- 等待信号
```

### 4. J线的特殊作用

```
J线的特点：
- 反应最快，最敏感
- 可以超过100或低于0
- 适合捕捉极端行情

J > 100：
- 极度超买
- 短期可能回调
- 但强势股可以J>120持续

J < 0：
- 极度超卖
- 短期可能反弹
- 但弱势股可以J<-10持续

用法：
J线是预警信号，K线是确认信号
```

### 5. 背离信号

**顶背离（卖出）**：
```
价格：           ↑ 创新高
KDJ：            ↗ ↘  未创新高
                  |
                 背离！可能下跌

要求：
- 价格创新高
- K值未创新高
- 在高位（K>60）
```

**底背离（买入）**：
```
价格：           ↘ 创新低
KDJ：            ↘ ↗  未创新低
                  |
                 背离！可能上涨

要求：
- 价格创新低
- K值未创新低
- 在低位（K<40）
```

### 6. 典型形态

| 形态 | KDJ特征 | 信号 |
|------|---------|------|
| 低位金叉 | K<20，K上穿D | 买入 |
| 高位死叉 | K>80，K下穿D | 卖出 |
| 底背离 | 价格新低，KDJ未新低 | 买入 |
| 顶背离 | 价格新高，KDJ未新高 | 卖出 |
| J线触底 | J<0后回升 | 短线买入 |
| J线触顶 | J>100后回落 | 短线卖出 |

### 7. KDJ与股价的关系

```
股价上升 + KDJ上升 = 健康上涨
股价上升 + KDJ下降 = 顶背离，警惕
股价下跌 + KDJ下跌 = 健康下跌
股价下跌 + KDJ上升 = 底背离，关注
```

## 四、A股实战要点

### 1. A股KDJ特色

| 特点 | 说明 | 影响 |
|------|------|------|
| T+1 | 当日不能卖出 | 金叉当日买入次日才能卖 |
| 涨跌停 | 涨停后KDJ可能失真 | 结合涨停数量 |
| 波动大 | KDJ变化快 | 需要更短周期 |
| 题材炒作 | 题材股KDJ持续高位 | 不要迷信数值 |

### 2. A股参数调整

```
大盘股：
- KDJ(9, 3, 3) 标准参数
- 稳定可靠

小盘股：
- KDJ(19, 3, 3) 或 KDJ(21, 3, 3)
- 更平滑，减少假信号

超短线：
- KDJ(5, 3, 3) 或 KDJ(6, 3, 3)
- 极敏感，捕捉日内机会

波段操作：
- KDJ(18, 3, 3) 或 KDJ(21, 3, 3)
- 减少噪音
```

### 3. A股实战策略

**策略1：低位金叉买入**
```
条件：
- KDJ低位（K<20）
- K线上穿D线
- 成交量放大
- 价格在支撑位附近

操作：
- 金叉当日尾盘买入
- 次日冲高卖出（T+1）
- 不破止损持有
```

**策略2：高位死叉卖出**
```
条件：
- KDJ高位（K>80）
- K线下穿D线
- 成交量萎缩
- 价格在阻力位附近

操作：
- 死叉当日减仓
- 次日反弹清仓
```

**策略3：J线触底反弹**
```
条件：
- J < 0
- J线开始回升
- K线也开始向上

操作：
- 超短线参与
- 快进快出
```

### 4. 注意事项

1. **高位钝化**：
   - 强势股可以KDJ长期>80
   - 不要因KDJ高位就卖出
   - 等死叉或趋势破坏

2. **低位钝化**：
   - 弱势股可以KDJ长期<20
   - 不要因KDJ低位就买入
   - 等金叉或趋势反转

3. **假金叉/死叉**：
   - 震荡市频繁出现
   - 必须等待确认（K线持续方向）
   - 结合趋势判断

4. **T+1限制**：
   - 当日金叉不能当日卖出
   - 需要提前判断次日走势
   - 或者在尾盘确认信号
