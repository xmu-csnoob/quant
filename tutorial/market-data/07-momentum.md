# 动量指标 (Momentum Indicators)

> 衡量价格变化速度和加速度的指标家族，捕捉市场"加速度"

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **速度测量**：价格上涨/下跌有多快？
- **加速度**：上涨/下跌在加速还是减速？
- **领先性**：动量变化通常领先于价格变化

### 解决什么问题
- 趋势还在加速还是已放缓？
- 价格创新高但动量减弱 = 背离
- 捕捉趋势反转的早期信号

### 适用场景
- 判断趋势强度
- 识别背离信号
- 捕捉超买超卖
- 确认突破有效性

### 指标家族

```
动量指标族：
├── 动量线 (Momentum)
├── ROC (变动率指标)
├── RSI (相对强弱指标)
├── 随机指标 (Stochastic/KDJ)
└── MACD (也有动量属性)

本文重点：Momentum 和 ROC
```

## 二、Momentum (动量线)

### 数学公式

```
Momentum = 今日价格 - N日前价格

标准参数：N = 10

示例：
M(10) = 今日收盘价 - 10日前收盘价
```

### 图形解读

```
价格持续上涨：
    Momentum = 正值且上升
    ↗↗↗

价格横盘：
    Momentum = 0附近波动
    →━→━→

价格持续下跌：
    Momentum = 负值且下降
    ↘↘↘
```

### 应用方法

| 信号 | 说明 | 操作 |
|------|------|------|
| Momentum > 0 | 近期上涨 | 偏多 |
| Momentum < 0 | 近期下跌 | 偏空 |
| Momentum 上升 | 上涨加速 | 持有 |
| Momentum 下降 | 上涨减速 | 警惕 |
| Momentum 背离 | 价格新高中未新高 | 可能反转 |

### 背离信号

```
顶背离：
价格：           ↑ 创新高
Momentum：       ↗ ↘  未创新高
                  |
                 警惕！

底背离：
价格：           ↘ 创新低
Momentum：       ↘ ↗  未创新低
                  |
                 关注！
```

## 三、ROC (Rate of Change)

### 数学公式

```
ROC = (今日价格 - N日前价格) / N日前价格 × 100%

标准参数：N = 12

示例：
ROC(12) = (今日收盘价 - 12日前收盘价) / 12日前收盘价 × 100%
```

### Momentum vs ROC

| 特性 | Momentum | ROC |
|------|----------|-----|
| 公式 | 绝对差值 | 相对百分比 |
| 单位 | 价格单位 | 百分比 |
| 适用范围 | 单个品种 | 可横向比较 |
| 优势 | 直观 | 可比较不同价位股票 |

### 图形解读

```
ROC = +10%  → 10日内上涨10%
ROC = 0     → 10日内持平
ROC = -5%   → 10日内下跌5%

ROC零轴：
  正值区 = 近期上涨
  负值区 = 近期下跌
```

### 应用方法

| ROC值 | 含义 | 操作思路 |
|-------|------|----------|
| ROC > +20% | 强势上涨 | 持有 |
| 0 < ROC < +20% | 温和上涨 | 观望 |
| -20% < ROC < 0 | 温和下跌 | 观望 |
| ROC < -20% | 强势下跌 | 观望/做空 |
| ROC 极端值 | 可能反转 | 关注 |

## 四、实战应用

### 1. 趋势强度判断

```
动量上升 + 价格上升 = 强势上涨趋势 → 持有
动量下降 + 价格上升 = 上涨乏力 → 警惕
动量上升 + 价格下降 = 下跌加速 → 观望
动量下降 + 价格下降 = 下跌放缓 → 关注底部
```

### 2. 背离交易

**顶背离（卖出信号）**：
```
步骤：
1. 价格创出新高
2. 动量指标未创新高
3. 等价格跌破支撑位确认
4. 卖出

示例：
价格：50 → 55 → 60 → 65（新高）
动量：+10 → +12 → +11 → +8（未新高）
→ 背离！警惕下跌
```

**底背离（买入信号）**：
```
步骤：
1. 价格创出新低
2. 动量指标未创新低
3. 等价格突破阻力位确认
4. 买入

示例：
价格：50 → 45 → 40 → 35（新低）
动量：-10 → -12 → -11 → -8（未新低）
→ 背离！关注反弹
```

### 3. 动量极值

```
动量极值含义：
ROC > +50%  → 极强势，警惕回调
ROC < -50%  → 极弱势，关注反弹

但注意：
强势股可以ROC长期高位
弱势股可以ROC长期低位
不要单凭极值操作
```

### 4. 动量与趋势结合

```
最佳买入信号：
1. 价格在MA之上（趋势向上）
2. 动量从负转正（动能转强）
3. 成交量放大

最佳卖出信号：
1. 价格在MA之下（趋势向下）
2. 动量从正转负（动能转弱）
3. 成交量萎缩
```

## 五、A股实战要点

### 1. A股动量特点

| 特点 | 说明 | 影响 |
|------|------|------|
| 涨跌停 | 价格受限，动量可能失真 | 结合涨停板数量 |
| T+1 | 当日无法卖出 | 动量信号提前判断 |
| 政策驱动 | 政策导致动量突变 | 注意政策影响 |
| 龙头股 | 龙头动量持续性强 | 区分龙头跟风 |

### 2. A股参数调整

```
大盘股：
- Momentum(10) 或 ROC(12)
- 波动小，标准参数

小盘股：
- Momentum(5) 或 ROC(9)
- 波动大，更敏感

题材股：
- Momentum(3) 或 ROC(5)
- 短期爆发，极敏感
```

### 3. A股背离实战

```
大盘背离：
上证指数创新高，ROC未创新高
→ 警惕大盘回调

个股背离：
龙头股背离 → 关注
跟风股背离 → 可能是假信号

板块背离：
板块指数背离 → 板块轮动信号
```

### 4. 注意事项

1. **涨停板影响**：
   - 连续涨停后动量指标可能钝化
   - 不要因动量高位而卖出强势股

2. **停牌影响**：
   - 复牌后动量可能异常
   - 剔除异常值

3. **除权除息**：
   - 价格调整导致动量失真
   - 使用复权价格计算

- 不要单凭动量极值操作，需结合趋势
