# 移动平均线 (Moving Average, MA)

> 最基础的技术分析工具，几乎所有指标的基础

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **平滑价格波动**：过滤日常噪音，显示价格主要趋势
- **趋势判断**：MA向上=上升趋势，MA向下=下降趋势
- **支撑压力**：价格常在MA附近获得支撑或遇到压力

### 解决什么问题
- 原始K线波动太大，难以看清趋势方向
- 需要一个"参照线"判断当前价格相对高低
- 需要判断趋势是否发生改变

### 适用场景
- 趋势行情（牛市/熊市）效果最好
- 震荡市中会产生频繁假信号
- 适合中长线投资参考

---

## 二、数学公式（如何计算？）

### 简单移动平均线 (SMA)

$$SMA_n = \frac{P_1 + P_2 + ... + P_n}{n}$$

其中：
- $P_i$ = 第i天的收盘价
- $n$ = 周期参数（常用5、10、20、60、120日）

### 指数移动平均线 (EMA)

$$EMA_t = \alpha \times P_t + (1-\alpha) \times EMA_{t-1}$$

其中：
- $\alpha = \frac{2}{n+1}$ = 平滑系数
- $P_t$ = 今日收盘价
- $EMA_{t-1}$ = 昨日EMA

### SMA vs EMA

| 特性 | SMA | EMA |
|------|-----|-----|
| 计算复杂度 | 简单 | 稍复杂 |
| 反应速度 | 慢 | 快 |
| 权重分配 | 所有价格等权重 | 近期价格权重高 |
| 适用场景 | 长期趋势判断 | 短期交易信号 |

### 计算示例

假设5日收盘价：[10, 11, 12, 13, 14]

```
SMA_5 = (10 + 11 + 12 + 13 + 14) / 5 = 12
```

EMA计算（假设α=0.4）：
```
EMA_1 = 10 (初始值)
EMA_2 = 0.4×11 + 0.6×10 = 10.4
EMA_3 = 0.4×12 + 0.6×10.4 = 11.04
EMA_4 = 0.4×13 + 0.6×11.04 = 11.82
EMA_5 = 0.4×14 + 0.6×11.82 = 12.69
```

## 三、应用（如何使用？）

### 1. 单根MA

| 信号 | 说明 |
|------|------|
| 价格在MA之上 | 多头信号，可能上涨 |
| 价格在MA之下 | 空头信号，可能下跌 |
| 价格突破MA | 趋势可能反转 |
| MA方向改变 | 趋势可能改变 |

### 2. 双MA交叉（金叉/死叉）

**金叉（买入信号）**：
- 短期MA从下向上穿越长期MA
- 示例：MA5上穿MA20

**死叉（卖出信号）**：
- 短期MA从上向下穿越长期MA
- 示例：MA5下穿MA20

### 3. 多MA排列

**多头排列**（强势）：
```
MA5 < MA10 < MA20 < MA60
所有MA向上，价格在MA5之上
```

**空头排列**（弱势）：
```
MA5 > MA10 > MA20 > MA60
所有MA向下，价格在MA5之下
```

### 4. 常用周期

| MA周期 | 名称 | 用途 |
|--------|------|------|
| MA5 | 周线 | 短线交易参考 |
| MA10 | 双周 | 短线趋势 |
| MA20 | 月线 | 中线波段（波段操作） |
| MA60 | 季线 | 中长线主力成本 |
| MA120 | 半年线 | 牛熊分界线 |
| MA250 | 年线 | 长期趋势判断 |

### A股实战注意事项

1. **T+1影响**：当日突破信号不能立即卖出
2. **涨跌停限制**：涨停后指标可能失真
3. **成交量配合**：MA突破需放量才可靠
4. **假突破**：震荡市中MA交叉频繁，容易反复打脸

### 常见陷阱

| 陷阱 | 说明 | 应对 |
|------|------|------|
| 滞后性 | MA是滞后指标，信号出现时趋势已运行一段时间 | 结合领先指标（如KDJ） |
| 震荡市失效 | 横盘时MA频繁交叉，产生假信号 | 震荡市避免使用MA |
| 参数敏感 | 不同周期信号可能矛盾 | 固定自己熟悉的周期组合 |
| 单指标依赖 | 只看MA容易受骗 | 结合成交量、MACD等 |
