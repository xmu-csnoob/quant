## 学习建议

### 1. 理解优先
先理解每个指标的**金融逻辑**，再记忆公式。问自己：
- 这个指标解决什么问题？
- 它的假设是什么？
- 什么时候会失效？

### 2. 结合实践
- 使用 TradingView 查看指标实际表现
- 用 Mock 数据测试指标计算
- 记录观察到的信号特征

### 3. 避免陷阱
- 单一指标不可靠，需要组合使用
- A股T+1规则影响指标有效性
- 指标信号存在滞后性

### 4. A股特色
- A股有涨跌停限制，影响指标形态
- 成交量在A股中尤为重要
- 政策因素会影响技术形态

## 常见问题

**Q: 需要数学基础吗？**
A: 基础的算术和理解百分比即可，每个文档都有计算示例。

**Q: 先学哪一个？**
A: 从移动平均线开始，它是多数指标的基础。

**Q: 这些指标都有效吗？**
A: 没有万能指标，关键是理解适用场景和局限性。
