## 一、股价形成的机制

### 没有公式，是竞价机制

```
拍卖会原理：
- 不是拍卖师喊价
- 是每个人自己喊"我愿意出多少钱买"或"我愿意多少钱卖"
- 当买家的出价和卖家的要价匹配时，交易就达成
- 最新成交价 = 当前股价
```

### 订单簿

```
卖单（Ask）          买单（Bid）
------------------------------
10.50元 [100手]      10.48元 [200手]
10.49元 [50手]       10.47元 [150手]
10.48元 [80手]  ←→   10.46元 [100手]

当前买卖价差（spread）= 10.48 - 10.46 = 0.02元
```

### 价格移动的原理

```
买盘强（买单多）：
- 10.48元的买单被吃完了
- 买家只能出到10.49元去吃上面的卖单
- 价格上涨到10.49元

卖盘强（卖单多）：
- 10.48元的卖单被抛完了
- 卖家只能挂10.47元去找下面的买单
- 价格下跌到10.47元
```

## 二、量价关系

### 成交量的定义

**成交量 = 实际成交的手数**

不是"挂单量"，而是"成交量"。

### 四种量价形态

#### 1. 量增价涨（健康上涨）

```
价格↑     成交量↑
    ↗
   /  放量上涨
  /

含义：买盘积极，资金流入，趋势健康
```

#### 2. 量增价跌（恐慌出货）

```
价格↓     成交量↑
  \
   \  放量下跌
    ↘

含义：恐慌性抛售，可能是底部（也可能是崩盘开始）
```

#### 3. 量减价涨（量价背离）

```
价格↑     成交量↓
        ↗
       /  缩量上涨
      /

含义：买盘不足，上涨乏力，危险信号
```

#### 4. 量减价跌（卖盘枯竭）

```
价格↓     成交量↓
      \
       \  缩量下跌
        ↘

含义：卖盘枯竭，可能接近底部
```

## 三、量价关系规律表

| 量价关系 | 市场含义 |
|---------|---------|
| **价涨量增** | 健康上涨，趋势可持续 |
| **价涨量减** | 上涨乏力，可能回调 |
| **价跌量增** | 恐慌抛售，注意风险 |
| **价跌量减** | 卖盘枯竭，可能见底 |

## 四、为什么成交量重要

### 核心原则

> **"价格是面子，成交量是里子"**

价格可以骗人（主力可以拉高价格），但成交量骗不了人（主力真金白银买入才会放量）。

### 成交量的优势

| 指标 | 能否造假 | 能否反映真实意图 |
|------|----------|-----------------|
| 价格 | 能（主力拉抬） | 不一定 |
| 成交量 | 难（需要真金白银） | 是 |

## 五、成交量指标

### 1. OBV（能量潮）- 最重要

**原理**：累加成交量，上涨日加成交量，下跌日减成交量

```python
if 今日收盘价 > 昨日收盘价:
    OBV = 昨日OBV + 今日成交量
elif 今日收盘价 < 昨日收盘价:
    OBV = 昨日OBV - 今日成交量
else:
    OBV = 昨日OBV
```

**用法**：
- OBV上涨 = 资金流入
- OBV下跌 = 资金流出
- OBV与价格背离 = 重要信号

**案例**：
```
价格：创新高 ✓
OBV：未创新高 ✗

→ 量价背离，上涨没有资金支撑，可能反转
```

### 2. 量比

**原理**：当日成交量 / 过去5日平均成交量

```
量比 > 3：放量
量比 < 0.5：缩量
```

### 3. 换手率

**原理**：成交量 / 流通股本

```
换手率 > 10%：活跃
换手率 < 2%：低迷
```

## 六、与前几个指标的关系

### 信息维度

```
价格 + 成交量 = 二维信息

但：
- MA、MACD、RSI 只用了价格 → 丢失了成交量信息
- 成交量是"真金白银"的方向，比价格更可靠
```

### 高级应用

- 用成交量确认价格趋势的真假
- 用成交量判断趋势的可持续性
- 用成交量识别主力资金流向
- 量价背离 = 反转信号

## 七、学习心得

### 思维提升

| 阶段 | 思维方式 |
|------|----------|
| 初级 | 看价格涨跌 |
| 中级 | 看技术指标 |
| **高级** | **看价格+成交量** |

### 核心收获

1. **股价不是算出来的，是竞价出来的**
   - 理解了订单簿机制
   - 理解了买卖盘失衡推动价格

2. **成交量比价格更可靠**
   - 价格可以造假
   - 成交量需要真金白银

3. **量价关系是市场心理的反映**
   - 量增价涨 = 众人追捧
   - 量增价跌 = 恐慌出逃
   - 量减价涨 = 上涨乏力
   - 量减价跌 = 卖盘枯竭

## 八、与实际策略的结合

### 策略改进方向

```
当前策略：MA + MACD + RSI（只用价格）

改进方向：加入成交量确认
- 金叉 + 放量 = 真突破
- 金叉 + 缩量 = 假突破
- 价格新高 + OBV新高 = 健康上涨
- 价格新高 + OBV不新高 = 背离，危险
```
