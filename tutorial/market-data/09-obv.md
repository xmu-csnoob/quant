# OBV (On-Balance Volume)

> 能量潮指标，由 Joe Granville 发明，将成交量转化为趋势指标

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **量是价的因**：资金流向先行于价格变化
- **累积能量**：OBV累积买卖盘力量
- **背离预警**：OBV背离常领先于价格反转

### 解决什么问题
- 主力资金是在买入还是卖出？
- 价格上涨是否有成交量支撑？
- 趋势是否健康（成交量配合）？

### 适用场景
- 判断主力资金动向
- 确认趋势有效性
- 捕捉背离信号
- 识别量价背离

### 指标形态

```
价格与OBV的关系：

健康上涨：          不健康上涨：
价格：  ↗↗↗        价格：  ↗↗↗
OBV：   ↗↗↗        OBV：   →→↘  ← 背离！

健康下跌：          不健康下跌：
价格：  ↘↘↘        价格：  ↘↘↘
OBV：   ↘↘↘        OBV：   →→↗  ← 背离！
```

## 二、数学公式（如何计算？）

### 计算步骤

**基础公式**：
```
如果今日收盘价 > 昨日收盘价：
    OBV = 昨日OBV + 今日成交量

如果今日收盘价 < 昨日收盘价：
    OBV = 昨日OBV - 今日成交量

如果今日收盘价 = 昨日收盘价：
    OBV = 昨日OBV
```

### 计算示例

```
天数 | 收盘价 | 成交量 | 价格变化 | OBV计算 | OBV值
-----|--------|--------|----------|---------|------
 1   | 10.00  | 1000   | -        | 0       | 0
 2   | 10.50  | 1500   | +0.50    | +1500   | 1500
 3   | 11.00  | 2000   | +0.50    | +2000   | 3500
 4   | 10.80  | 1800   | -0.20    | -1800   | 1700
 5   | 10.60  | 1200   | -0.20    | -1200   | 500
 6   | 11.00  | 2500   | +0.40    | +2500   | 3000
 7   | 11.50  | 3000   | +0.50    | +3000   | 6000

解读：
- 2-3天：量价齐升，OBV上升（健康）
- 4-5天：价格下跌，OBV下降（正常）
- 6-7天：量价齐升，OBV上升（健康）
```

### OBV变体

**1. OBV均线**：
```
OBV_MA = MA(OBV, n)

用途：
- OBV金叉OBV_MA = 买盘增强
- OBV死叉OBV_MA = 卖盘增强
```

**2. OBV变化率**：
```
OBV_ROC = (今日OBV - N日前OBV) / N日前OBV × 100%

用途：
- 判断量能变化速度
```

## 三、应用（如何使用？）

### 1. 趋势确认

| 情况 | OBV状态 | 趋势健康度 |
|------|---------|------------|
| 价格上涨 | OBV上升 | 健康，可能继续 |
| 价格上涨 | OBV横盘 | 乏力，警惕 |
| 价格上涨 | OBV下降 | 背离，可能反转 |
| 价格下跌 | OBV下降 | 健康，可能继续 |
| 价格下跌 | OBV横盘 | 跌势放缓 |
| 价格下跌 | OBV上升 | 背离，可能反转 |

### 2. OBV背离（核心应用）

**顶背离（卖出信号）**：
```
价格：           ↑ 创新高
OBV：            ↗ → ↘  未创新高
                      |
                     背离！主力在出货

含义：
- 价格创新高
- 但买盘不增反减
- 主力可能在出货
→ 考虑卖出
```

**底背离（买入信号）**：
```
价格：           ↘ 创新低
OBV：            ↘ → ↗  未创新低
                      |
                     背离！主力在吸筹

含义：
- 价格创新低
- 但卖压减轻
- 主力可能在吸筹
→ 考虑买入
```

### 3. OBV突破

```
OBV突破前高：
- 买盘增强
- 可能新一轮上涨

OBV跌破前低：
- 卖压增大
- 可能新一轮下跌

示例：
OBV:  →━→━→━→━━↗  突破前高
价格： →→→→↗  配合突破
→ 买入信号
```

### 4. OBV与价格配合

| 价格 | OBV | 市场含义 | 操作 |
|------|-----|----------|------|
| ↗ 上升 | ↗ 上升 | 健康上涨 | 持有 |
| ↗ 上升 | → 横盘 | 上涨乏力 | 减仓 |
| ↗ 上升 | ↘ 下降 | 顶背离 | 卖出 |
| ↘ 下跌 | ↘ 下降 | 健康下跌 | 观望 |
| ↘ 下跌 | → 横盘 | 下跌放缓 | 关注 |
| ↘ 下跌 | ↗ 上升 | 底背离 | 买入 |

### 5. OBV形态分析

OBV也可以形成各种形态：

| 形态 | 含义 | 信号 |
|------|------|------|
| 双底 | OBV两次探底回升 | 买入 |
| 双顶 | OBV两次冲顶回落 | 卖出 |
| 上升三角形 | OBV高位震荡向上 | 继续持有 |
| 下降三角形 | OBV低位震荡向下 | 观望 |
| 突破平台 | OBV突破盘整区 | 趋势开始 |

### 6. OBV极值

```
OBV历史新高：
- 买盘累积到极致
- 可能持续上涨
- 或临近顶部

OBV历史新低：
- 卖盘累积到极致
- 可能持续下跌
- 或临近底部

关键：看是否背离
```

## 四、A股实战要点

### 1. A股OBV特色

| 特点 | 说明 | 影响 |
|------|------|------|
| T+1 | 当日买入次日才能卖 | OBV信号有延迟 |
| 涨跌停 | 涨停后OBV可能失真 | 结合涨停数量 |
| 机构资金 | 机构资金影响大 | OBV反映机构动向 |
| 融资融券 | 杠杆资金影响 | OBV可能放大 |

### 2. A股实战策略

**策略1：OBV背离交易**
```
条件：
- 价格创新高/新低
- OBV未创新高/新低
- 等待反转K线确认

操作：
- 顶背离：减仓
- 底背离：建仓
```

**策略2：OBV突破交易**
```
条件：
- OBV突破平台
- 价格配合突破
- 成交量放大

操作：
- 突破当日跟进
- 止损设置在突破前
```

**策略3：OBV与主力资金**
```
条件：
- 价格横盘
- OBV持续上升
- 可能是主力吸筹

操作：
- 低位买入
- 等待突破
```

### 3. A股注意事项

1. **涨跌停影响**：
   - 涨停后成交量急剧萎缩
   - OBV可能不再上升（但正常）
   - 不要因OBV横盘认为量价背离

2. **新股上市**：
   - 初期成交量异常大
   - OBV失真
   - 需要一段时间的稳定数据

3. **除权除息**：
   - 价格调整不影响OBV
   - 但成交量可能异常
   - 需要修正

4. **停牌**：
   - 停牌期间OBV不变
   - 复牌后可能跳变
   - 需要平滑处理

### 4. 常见陷阱

| 陷阱 | 说明 | 应对 |
|------|------|------|
| 涨停失真 | 涨停后OBV横盘但正常 | 看涨停数量和封单 |
| 无量上涨 | 价格涨OBV不动 | 可能是拉高出货 |
| 巨量震荡 | OBV大幅波动 | 等待趋势明确 |
| 假突破 | OBV突破后快速回落 | 等确认信号 |

**要点总结**：
- OBV累积涨跌成交量，反映资金流向
- OBV上升=资金流入，OBV下降=资金流出
- OBV背离是重要的预警信号
- 量价配合是健康趋势的基础
- A股注意涨跌停对OBV的影响
- 结合价格趋势判断更可靠
