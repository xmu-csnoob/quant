# 成交量指标 (Volume Indicators)

> 成交量是价格的燃料，量价配合分析是技术分析的核心

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **量是价的因**：成交量变化通常领先于价格变化
- **量价配合**：有效突破需要放量，无量为假
- **市场能量**：成交量反映市场参与度和资金流向

### 解决什么问题
- 价格突破是真是假？
- 趋势是否有持续力？
- 主力资金在买入还是卖出？

### 适用场景
- 确认突破信号的真假
- 判断趋势是否健康
- 捕捉主力资金动向
- 识别顶部和底部

### 核心原则

```
量价关系口诀：

量增价涨 → 健康上涨，可能继续
量减价涨 → 上涨乏力，警惕回调
量增价跌 → 恐慌下跌，可能继续
量减价跌 → 卖压减轻，可能反弹

量价背离 → 趋势可能反转
无量突破 → 假突破概率高
```

## 二、基础成交量分析

### 1. 量价关系

| 情况 | 含义 | 后续预期 |
|------|------|----------|
| **量增价涨** | 量价齐升，多方强 | 继续上涨概率大 |
| **量减价涨** | 上涨乏力，量能不足 | 可能回调 |
| **量增价跌** | 恐慌抛售，空方强 | 可能继续下跌 |
| **量减价跌** | 卖压减轻 | 可能触底反弹 |

### 2. 放量与缩量

```
放量：
- 成交量 > 平均成交量的1.5倍
- 含义：资金积极进出
- 应用：确认突破、判断顶底

缩量：
- 成交量 < 平均成交量的0.7倍
- 含义：市场观望情绪
- 应用：判断趋势衰竭
```

### 3. 天量与地量

```
天量（极端放量）：
- 成交量创历史新高
- 含义：见顶信号或换手充分
- 应用：顶部减仓

地量（极端缩量）：
- 成交量创历史新低
- 含义：抛压枯竭
- 应用：底部区域
```

## 三、常用成交量指标

### 1. OBV (On-Balance Volume)

**原理**：累积涨跌成交量

```
如果今日上涨：OBV = 昨日OBV + 今日成交量
如果今日下跌：OBV = 昨日OBV - 今日成交量
如果今日平盘：OBV = 昨日OBV
```

**应用**：
- OBV上升 = 买盘累积，看涨
- OBV下降 = 卖盘累积，看跌
- OBV背离 = 价格新高但OBV未创新高，警惕

**详细学习**：[09. OBV](./09-obv.md)

### 2. 成交量移动平均 (VMA)

```
VMA(5) = 5日成交量均值
VMA(20) = 20日成交量均值
```

**应用**：
- VMA金叉 = 量能放大
- VMA死叉 = 量能萎缩
- 成交量突破VMA = 量能突变

### 3. 量比 (Volume Ratio)

```
量比 = (当前成交量) / (过去5日平均同期成交量)

量比 > 2 = 明显放量
量比 > 5 = 剧烈放量
量比 < 0.5 = 明显缩量
```

### 4. 换手率 (Turnover Rate)

```
换手率 = 成交量 / 流通股本 × 100%

换手率等级：
- < 3%：交投清淡
- 3-7%：交投活跃
- 7-10%：高度活跃
- > 10%：极度活跃（警惕）
- > 20%：可能顶部
```

## 四、实战应用

### 1. 突破确认

```
真突破：
价格突破 + 成交量放大 + 持续放量

假突破：
价格突破 + 成交量萎缩 或 一日放量后缩量

示例：
股价突破前高，成交量 > 1.5倍VMA
次日继续放量 → 真突破
次日缩量 → 假突破
```

### 2. 底部识别

```
底部特征：
1. 价格长期下跌
2. 成交量持续萎缩（地量）
3. 出现放量拉升
4. 之后缩量不创新低

图形：
    ↘↘↘ 价格
      ━━━━━ 成交量（地量）
      ↗ 突然放量
    ↗ 开始上涨
```

### 3. 顶部识别

```
顶部特征：
1. 价格长期上涨
2. 成交量异常放大（天量）
3. 价格滞涨或下跌
4. 之后量价背离

图形：
    ↗↗↗ 价格
      ━━━━━━━━━ 成交量（天量）
      ↘ 开始下跌
```

### 4. 主力资金分析

```
主力建仓：
- 价格横盘
- 成交量温和放大
- OBV缓慢上升

主力出货：
- 价格上涨
- 成交量异常放大
- OBV不创新高（背离）
```

## 五、A股实战要点

### 1. A股成交量特色

| 特点 | 说明 | 影响 |
|------|------|------|
| T+1 | 当日买入次日才能卖 | 成交量有滞后性 |
| 涨跌停 | 涨停后成交量可能急剧萎缩 | 失真的量能 |
| 机构投资者 | 机构资金量大 | 机构动向影响大 |
| 政策市 | 政策影响成交量 | 突发放量 |

### 2. A股量价关系

```
牛市：
- 量增价涨 = 正常
- 量减价涨 = 谨慎（可能顶部）

熊市：
- 量增价跌 = 恐慌（可能底部）
- 量减价跌 = 阴跌（继续下跌）

A股特有：
- 放量涨停 = 强势，可能连续涨停
- 跌停开板放量 = 可能反弹
```

### 3. A股成交量参数

```
大盘股：
- 正常日成交量：50-200亿
- 放量标准：>1.5倍VMA
- 天量标准：>2倍VMA

小盘股：
- 正常日成交量：5-20亿
- 放量标准：>2倍VMA
- 天量标准：>3倍VMA
```

### 4. 注意事项

1. **涨跌停失真**：
   - 涨停后成交量会锐减
   - 不要因缩量认为动能不足
   - 看封单大小判断强度

2. **新股特殊**：
   - 新股上市初期成交量巨大
   - 不要与正常时期比较
   - 换手率>50%需警惕

3. **季报/年报期**：
   - 业绩前后成交量放大
   - 可能是内幕交易
   - 谨慎跟随

## 速查表

| 量价关系 | 含义 | 操作 |
|----------|------|------|
| 量增价涨 | 健康上涨 | 持有/加仓 |
| 量减价涨 | 上涨乏力 | 减仓/观望 |
| 量增价跌 | 恐慌下跌 | 观望/做空 |

| 无量突破 | 假突破概率高 | 不操作 |
| 放量突破 | 真突破概率高 | 顺势跟进 |
