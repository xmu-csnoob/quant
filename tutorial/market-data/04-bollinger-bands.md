# 布林带 (Bollinger Bands)

> 由 John Bollinger 发明，结合趋势和波动率的动态指标，能自适应市场变化

---

## 一、原理（为什么需要它？）

### 金融逻辑
- **波动率测量**：带宽反映市场波动程度
- **动态支撑压力**：上下轨随波动率自动调整
- **均值回归**：价格大概率在带内波动，触及边界会回归

### 解决什么问题
- 固定宽度的通道（如包络线）无法适应不同市场状态
- 需要判断价格是否"偏离正常范围"
- 需要判断波动率是收缩（爆发前）还是扩张（趋势中）

### 适用场景
- 趋势突破：布林带收缩后扩张，往往是大行情起点
- 震荡高抛低吸：触及上下轨做反向操作
- 趋势跟踪：价格沿上轨运行=强势，沿下轨运行=弱势

### 指标形态

```
上轨 = MA + 2倍标准差 ← 动态压力线
─────────────────────────────────
      |    |    |    |    |   ← 正常波动区域
中轨 = MA(20)             ← 趋势方向
─────────────────────────────────
      |    |    |    |    |
下轨 = MA - 2倍标准差 ← 动态支撑线

收缩 ︎  ↕  扩张
[===]    [            ]
```

## 二、数学公式（如何计算？）

### 计算步骤

**1. 计算中轨（Middle Band）**
```
中轨 = SMA(收盘价, 20)
即20日简单移动平均线
```

**2. 计算标准差（Standard Deviation）**
```
σ = √[Σ(价格 - 均值)² / n]

其中：
- 价格 = 每日收盘价
- 均值 = 20日均价（中轨）
- n = 20（周期）
```

**3. 计算上轨（Upper Band）**
```
上轨 = 中轨 + (k × σ)

标准参数：k = 2
```

**4. 计算下轨（Lower Band）**
```
下轨 = 中轨 - (k × σ)
```

### 参数说明

| 参数 | 标准值 | 可调范围 | 说明 |
|------|--------|----------|------|
| 周期 | 20 | 10-50 | 越小越敏感 |
| 标准差倍数 | 2 | 1.5-2.5 | 越大通道越宽 |
| 均线类型 | SMA | SMA/EMA | 一般用SMA |

### 为什么是2倍标准差？

根据正态分布：
- ±1σ 包含 68.3% 的数据
- ±2σ 包含 95.4% 的数据
- ±3σ 包含 99.7% 的数据

**选择2σ的含义**：
- 95.4%的时间价格会在带内
- 触及边界是极端情况（概率<5%）
- 可能发生回归或突破

### 计算示例

假设某股票10天数据（简化为10天）：

```
天数 | 收盘价 | 20日均价 | 与均值差 | 差的平方
-----|--------|---------|---------|----------
 1   | 10.00  | 10.20   | -0.20   | 0.04
 2   | 10.50  | 10.20   | +0.30   | 0.09
 3   | 11.00  | 10.20   | +0.80   | 0.64
 4   | 10.80  | 10.20   | +0.60   | 0.36
 5   | 10.20  | 10.20   | 0       | 0
 ... | ...    | ...     | ...     | ...
20   | 10.50  | 10.20   | +0.30   | 0.09

计算：
方差 = Σ(差的平方) / 20 = 3.5 / 20 = 0.175
标准差 σ = √0.175 ≈ 0.42

上轨 = 10.20 + (2 × 0.42) = 11.04
中轨 = 10.20
下轨 = 10.20 - (2 × 0.42) = 9.36
```

## 三、应用（如何使用？）

### 1. 基础信号

| 信号 | 说明 | 操作 |
|------|------|------|
| 价格触及上轨 | 超买，可能回调 | 考虑卖出 |
| 价格触及下轨 | 超卖，可能反弹 | 考虑买入 |
| 价格突破上轨 | 极强势，可能继续 | 持有/加仓 |
| 价格跌破下轨 | 极弱势，可能继续 | 观望/做空 |
| 价格在中轨附近 | 正常波动 | 观望 |

### 2. 布林带收缩（Squeeze）

**收缩含义**：波动率降到极低，能量积聚

```
收缩前：                收缩中：
  ┌─────────┐           ┃━┃
  │         │           ┃━┃
  │         │    →      ┃━┃  ← 波动率极低！
  │         │           ┃━┃
  └─────────┘           ┃━┃

然后可能爆发：
上方向突破 → 上涨趋势
下方向突破 → 下跌趋势
```

**操作策略**：
1. 识别收缩（带宽变窄）
2. 等待突破方向确认
3. 顺势跟进

### 3. 布林带扩张（Expansion）

**扩张含义**：波动率放大，趋势进行中

```
收缩后的扩张：
    ↗ ↗ ↗ ↗ ↗ ↗
   ┌             ┐
   │             │
   │             │  ← 趋势确立
   │             │
   └             ┘
    ↘ ↘ ↘ ↘ ↘ ↘

策略：顺势而为，不要逆势
```

### 4. 价格沿轨道运行

**强势特征**：
```
价格沿上轨运行 → 极强势
  ┌─────────────────────────↗
  │                      ↗
  │                   ↗
  │                ↗
  │             ↗
  └─────────────

操作：持有，不轻易卖出
```

**弱势特征**：
```
价格沿下轨运行 → 极弱势
          ↘ ┌────────────────
        ↘   │
      ↘     │
    ↘       │
  ↘         │
───────────┘

操作：观望或做空
```

### 5. 布林带宽度（Bandwidth）

```
带宽 = (上轨 - 下轨) / 中轨

应用：
- 带宽 < 极小值 → 收缩，准备突破
- 带宽突然增大 → 突破发生
- 带宽持续高位 → 趋势行情
```

### 6. %B 指标（位置指标）

```
%B = (价格 - 下轨) / (上轨 - 下轨)

值域：
%B = 1  → 价格在上轨
%B = 0  → 价格在下轨
%B = 0.5 → 价格在中轨
%B > 1  → 价格突破上轨
%B < 0  → 价格跌破下轨

应用：
- %B > 1 → 超买
- %B < 0 → 超卖
```

### 7. 典型形态

| 形态 | 图形 | 信号 |
|------|------|------|
| 双顶（M头） | 价格两次触及上轨 | 卖出 |
| 双底（W底） | 价格两次触及下轨 | 买入 |
| 收缩突破 | 带宽突然放大 | 顺势跟进 |
| 沿上轨运行 | 持续贴近上轨 | 强势持有 |

### A股实战注意事项

1. **T+1影响**：
   - 触及上轨当天不能卖出
   - 需要提前预判：接近上轨时考虑减仓

2. **涨跌停影响**：
   - 涨停后价格可能持续压在上轨
   - 布林带失去参考意义

3. **A股特色**：
   - 龙头股可以长期沿上轨运行
   - 突破上轨后可能连续涨停
   - 政策利好会导致瞬间扩张

4. **参数调整**：
   - 大盘股：标准参数(20, 2)
   - 小盘股：(15, 2.5) 更宽
   - 高波动股：(10, 3) 捕捉极端行情

### 常见陷阱

| 陷阱 | 说明 | 应对 |
|------|------|------|
| 假突破 | 价格短暂突破轨道又回归 | 等确认（收盘价站稳） |
| 收缩假突破 | 收缩后小幅震荡再反方向 | 等待明显突破信号 |
| 强势钝化 | 牛股长期沿上轨运行 | 不要轻易做空 |
| 单一指标依赖 | 只看布林带容易被骗 | 结合成交量、MACD |
