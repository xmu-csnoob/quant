# 技术指标：移动平均线（MA）

## 什么是移动平均线？

**移动平均线（Moving Average，简称 MA）** 是最简单的技术指标。

**核心思想：** 把过去 N 天的收盘价取平均，得到一个"平滑的价格"。

## 为什么需要 MA？

**原始价格太"吵"**
```
收盘价：1850 → 1888 → 1905 → 1880 → 1915 → 1890 → ...
```
价格上下波动，很难看出趋势。

**MA 让趋势更清晰**
```
MA5：  1860 → 1870 → 1880 → 1885 → 1890 → 1892 → ...
```
平滑后的线，趋势一目了然。

## 如何计算 MA5（5日均线）？

```
第5天  MA5 = (Day1+Day2+Day3+Day4+Day5) / 5
第6天  MA5 = (Day2+Day3+Day4+Day5+Day6) / 5
第7天  MA5 = (Day3+Day4+Day5+Day6+Day7) / 5
...
```

"移动"的意思是：每天往后面挪一天，重新计算。

## 用示例数据计算 MA5

| 日期 | 收盘价 | MA5计算 | MA5值 |
|------|--------|---------|-------|
| 01-02 | 1875 | - | - |
| 01-03 | 1888 | - | - |
| 01-04 | 1905 | - | - |
| 01-05 | 1915 | - | - |
| 01-08 | 1935 | (1875+1888+1905+1915+1935)/5 | 1903.6 |
| 01-09 | 1945 | (1888+1905+1915+1935+1945)/5 | 1917.6 |
| 01-10 | 1958 | (1905+1915+1935+1945+1958)/5 | 1931.6 |
| 01-11 | 1962 | (1915+1935+1945+1958+1962)/5 | 1943.0 |
| 01-12 | 1975 | (1935+1945+1958+1962+1975)/5 | 1955.0 |

## MA 的用途

**1. 判断趋势**
- 价格在 MA 之上 → 上涨趋势
- 价格在 MA 之下 → 下跌趋势

**2. 金叉死叉**
- MA5 上穿 MA20 → 金叉（买入信号）
- MA5 下穿 MA20 → 死叉（卖出信号）

**3. 支撑压力**
- 价格回落到 MA 附近 → 可能获得支撑
- 价格上涨到 MA 附近 → 可能遇到压力

## 常用的 MA 周期

| 周期 | 名称 | 用途 |
|------|------|------|
| MA5 | 短期 | 捕捉短期趋势 |
| MA10 | 短期 | 日内交易常用 |
| MA20 | 中期 | 月线趋势 |
| MA60 | 中期 | 季度趋势 |
| MA120 | 长期 | 半年趋势 |
| MA250 | 长期 | 年线趋势（牛熊分界） |

## Python 实现

```python
import pandas as pd

# 读取数据
df = pd.read_csv('sample_data.csv')

# 计算 MA5
df['MA5'] = df['收盘'].rolling(window=5).mean()

# 计算 MA20
df['MA20'] = df['收盘'].rolling(window=20).mean()
```
