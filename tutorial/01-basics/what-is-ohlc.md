# 量化交易基础：什么是 OHLC 数据

## 股票数据的最基本形式

量化交易的第一步是理解股票数据。股票数据最基本的格式是 **OHLC**：

| 字母 | 含义 | 说明 |
|------|------|------|
| O | Open | 开盘价 |
| H | High | 最高价 |
| L | Low | 最低价 |
| C | Close | 收盘价 |

通常还会包含：
- **Volume**：成交量
- **Date**：日期

## 示例数据

假设我们要分析贵州茅台（600519.SH）2024年1月的某几天数据：

```
日期          开盘    最高    最低    收盘    成交量
2024-01-02   1850.00 1880.00 1840.00 1875.00  2.5M
2024-01-03   1872.00 1895.00 1860.00 1888.00  3.1M
2024-01-04   1885.00 1910.00 1880.00 1905.00  2.8M
```

## 为什么用 OHLC？

**单价格不够用**
- 只看收盘价，你会错过当天的波动
- 当天可能在1850和1910之间大幅震荡，收盘价无法体现

**OHLC 能告诉你什么**
- 当天波动范围：H - L
- 收盘相对于开盘的涨跌：C - O
- 是否在高位收盘：C 靠近 H 还是 L？

## A股的特点

**T+1 制度**
- 今天买入的股票，明天才能卖出
- 影响策略：不能日内频繁交易

**涨跌停限制**
- 主板：±10%
- 创业板/科创板：±20%
- 北交所：±30%

**交易时间**
- 上午：9:30-11:30
- 下午：13:00-15:00
